# ESP32-C6 + ELM327 + A7670E для Home Assistant

Цей проєкт стартує з надійної частини: ESP32-C6 читає OBD-II через ELM327 по UART і віддає сенсори в Home Assistant через ESPHome API/Wi-Fi.

## Важливе обмеження

ELM327 має Bluetooth Classic SPP-модуль. ESP32-C6 підтримує BLE, але не підтримує Bluetooth Classic/SPP, тому напряму з'єднатися з цим ELM327 по Bluetooth не вийде.

Потрібний варіант: підпаятися до UART між основною платою ELM327 та Bluetooth-модулем.

## Підключення

Спільне:

- GND ESP32-C6 -> GND ELM327
- ELM327 живиться від OBD-II роз'єму автомобіля
- ESP32-C6 краще живити від окремого стабілізованого 5 V buck-конвертера з ACC/12 V, не від випадкової точки на ELM327

UART ELM327:

- TX ELM327 -> RX ESP32-C6, у YAML це `GPIO17`
- RX ELM327 -> TX ESP32-C6, у YAML це `GPIO16`
- Рівень має бути 3.3 V TTL. Перед підключенням перевір мультиметром або логічним аналізатором.

На багатьох платах ELM327 UART-піни видно на місці, де модуль `BT-05/CY-BT-05` припаяний до плати. Зазвичай треба знайти `TXD`, `RXD`, `GND`, `VCC`. Якщо написів немає, зручно продзвонити доріжки від Bluetooth-модуля.

## A7670E-MASA + GNSS

A7670E можна додати другим UART:

- TX A7670E -> RX ESP32-C6
- RX A7670E -> TX ESP32-C6
- GND спільний
- живлення модуля має витримувати піки струму LTE, зазвичай 2 A або більше

У цьому стартовому YAML LTE-канал не ввімкнений як мережа ESPHome. ESPHome стабільно працює через Wi-Fi/API. Для A7670E є два практичні наступні кроки:

- GNSS: увімкнути NMEA на модемі AT-командами і парсити координати окремим ESPHome-компонентом.
- LTE телеметрія: зробити окремий компонент, який через AT-команди публікує MQTT/HTTP напряму в Home Assistant або брокер.

## Файли

- `car_obd.yaml` - основний ESPHome конфіг.
- `components/elm327_uart/` - локальний компонент для читання OBD-II PID через UART.

## Перший запуск

1. У `car_obd.yaml` заміни `YOUR_WIFI_SSID`, `YOUR_WIFI_PASSWORD` і ключі.
2. Перевір GPIO для своєї ESP32-C6 SuperMini. Якщо `GPIO20/GPIO21` зайняті або незручні, зміни їх у секції `uart`.
3. Проший ESP32-C6 через ESPHome.
4. Підключи ELM327 до авто, увімкни запалювання.
5. У логах ESPHome має з'явитися ініціалізація ELM327 і значення сенсорів.

## Що вже читається

- RPM
- швидкість
- температура охолоджувальної рідини
- напруга ECU/бортмережі через PID `0142`
- положення дроселя
- температура впускного повітря

Не всі авто підтримують усі PID. Якщо сенсор лишається `unknown`, це може бути нормально для конкретної ECU.
